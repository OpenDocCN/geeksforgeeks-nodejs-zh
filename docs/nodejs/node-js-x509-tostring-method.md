# Node.js x509.toString()方法

> 原文:[https://www.geeksforgeeks.org/node-js-x509-tostring-method/](https://www.geeksforgeeks.org/node-js-x509-tostring-method/)

**x509.toString()** 是加密模块中 x509 证书类的内置应用编程接口，用于获取 PEM 编码的证书。

**语法:**

```
const x509.toString()
```

**参数:**此方法不接受任何参数作为参数。

**返回值:**该方法返回 PEM 编码的证书。

**如何生成公共证书？**

**例 1:**

## index . js

```
// Node.js program to demonstrate the  
// x509.toString() function

// Importing crypto module
const {X509Certificate} = require('crypto')

// Importing fs module
const fs = require('fs')

// getting object of a PEM encoded X509 Certificate. 
const x509 = new X509Certificate(fs.readFileSync('public-cert.pem'));

// getting string representation of this certificate.
// by using x509.toString() function
const value = x509.toString()

// display the result
console.log( value)
```

使用以下命令运行 index.js 文件。

```
node index.js
```
